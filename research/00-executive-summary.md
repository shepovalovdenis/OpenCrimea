# Executive Summary: OpenCrimea
## Open-source платформа аренды жилья в Крыму

---

## 1. ВЕРДИКТ — GO ✅

**Проект стоит реализовывать.** OpenCrimea имеет четкую стратегию выхода на нишевый рынок с сильными конкурентными преимуществами: 0% комиссии vs 12-18% у конкурентов, open-source прозрачность и глубокая локальная специализация. При базовом сценарии проект достигает безубыточности через 13 месяцев и показывает EBITDA 41,7 млн руб к 3-му году при инвестициях всего 1,5 млн руб.

## 6. ЮРИДИКА

**Рекомендуемая структура:**
- **Форма:** ИП в Крыму
- **Налоговый режим:** УСН "Доходы" **4%** (льготная ставка до 2028)
- **Локализация данных:** Обязательная по 152-ФЗ (персональные данные в РФ)
- **Платежи:** СБП, Мир Pay, ЮMoney, банковские карты РФ

**Основные ограничения:**
- Геоблокировка для санкционных стран
- Compliance с российским налоговым и валютным законодательством
- Требования к защите персональных данных

**Документооборот:**
- Политика конфиденциальности
- Пользовательское соглашение
- Согласия на обработку ПДн
- Договор-оферта на услуги

## 7. ТЕХНИЧЕСКАЯ СТРАТЕГИЯ

**Tech Stack (5 ключевых решений):**

1. **Architecture:** Модульный монолит → микросервисы при масштабировании
2. **Frontend:** Next.js + TypeScript + Tailwind (современный, SEO-friendly)
3. **Backend:** Fastify + PostgreSQL + Redis (производительность + надежность)  
4. **Infrastructure:** Yandex Cloud (российский провайдер, санкционная устойчивость)
5. **Mobile:** PWA сначала, затем React Native для российских сторов

**DevOps:** Docker + Kubernetes + GitHub Actions + мониторинг (Prometheus + Grafana)

**Безопасность:** Шифрование ПДн + JWT + RBAC + регулярные аудиты безопасности

## 8. OPEN-SOURCE СТРАТЕГИЯ

**Модель лицензирования:**
- **Core Platform:** AGPL v3 (защищает от проприетарных форков)
- **Premium Modules:** Proprietary (AI-аналитика, автоценообразование)
- **Лицензия:** AGPL v3 рекомендуется

**Монетизация Open-Core:**

| Сегмент | Что включено | Цена |
|---------|-------------|------|
| **Community** | Core функции, базовая поддержка | Бесплатно |
| **Premium** | Расширенная аналитика, AI-рекомендации | $9-29/мес |
| **Enterprise** | White-label, on-premise, dedicated support | $99-499/мес |

**Community building:** Discord + GitHub + bounty система ($50-2000 за задачи) + программа амбассадоров

**Конкурентное преимущество:** Прозрачность алгоритмов vs "черные ящики" конкурентов
