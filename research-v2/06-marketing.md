**Источники данных:**

- **Суточно.ру:** премиум категории, извлечение владельцев
- **Instagram:** хэштеги #крымаренда #премиумотдых #villacrimea
- **VK:** группы аренды, поиск по ключевым словам "вилла", "коттедж", "премиум"

**Техническая реализация:**
```python
# Псевдокод автоматического парсинга
def parse_premium_hosts():
    targets = []
    
    # Авито API/Selenium
    avito_listings = parse_avito(min_price=5000, property_type="house")
    
    # Извлечение контактов
    for listing in avito_listings:
        contact_info = extract_contact(listing)
        targets.append({
            'phone': contact_info.phone,
            'telegram': find_telegram_by_phone(contact_info.phone),
            'whatsapp': contact_info.phone,
            'property_data': listing.details
        })
    
    return targets
```

## 📊 Метрики для премиум-сегмента

### 1. Качественные метрики вместо количественных
**Хосты:**
- Average Property Value: >5,000₽/сутки (вместо общего количества)
- Host LTV: средний доход хоста за год
- Premium Host Retention: % остающихся хостов высокого уровня
- Quality Score: рейтинг объектов (только 4.5+ звезд)

**Гости:**
- Customer LTV: средний lifetime value премиум гостя  
- Repeat Premium Rate: % повторных бронирований 5К+
- Average Booking Value: средний чек бронирования
- Net Promoter Score: только для премиум гостей (цель: 70+)

### 2. Маркетинговая эффективность
**Общие:**
- CAC Payback: окупаемость привлечения премиум клиента
- ROAS by Channel: доходность каналов для премиум сегмента  
- Viral Coefficient: сколько рефералов приводит 1 премиум пользователь
- Market Share: доля в премиум сегменте Крыма

**Цели на 2026:**
- **Q1:** 25 премиум хостов (5К+ сутки), 100 VIP гостей
- **Q2:** 50 премиум хостов, 300 VIP гостей, break-even
- **Q3:** 75 хостов, 500 VIP гостей, 2x ROI
- **Q4:** 100 хостов, 1000 VIP гостей, market leadership
